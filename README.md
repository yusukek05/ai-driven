# GPT-2 Chatbot

A simple text chatbot powered by OpenAI's GPT-2, built with Streamlit.

## Models

| Model | Size |
|-------|------|
| gpt2 small | 124M |
| gpt2 medium | 355M |

## Run locally

```bash
uv run streamlit run app.py
```

## Deploy

Deployed on [Streamlit Community Cloud](https://streamlit.io/cloud).
